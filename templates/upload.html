Инструкции по загрузке

Порядок обработки:
  1. Система автоматически определит тип каждого файла
  2. Данные будут импортированы в базу данных
  3. Запустится процесс интеллектуальной сверки
  4. Результаты будут сохранены и отправлены в Telegram
Типы ошибок сверки:
  • OK - Всё совпадает
  • NO_MATCH_IN_REPORT - Нет во внутреннем учете
  • NO_PAYMENT_FOUND - Нет оплаты
  • FISCAL_MISSING - Нет фискального чека
